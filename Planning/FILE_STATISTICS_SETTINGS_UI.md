## 📞 技术支持

### 常见问题解答

**Q: 为什么需要 API Key？**
A: 用于验证用户身份，防止滥用，OpenAI 按使用量计费需要追踪用户。

**Q: 本地后端有什么优势？**
A: 开发测试快速，无网络延迟，免费测试，不消耗 API 配额。

**Q: 如何更换 API 提供商？**
A: 打开设置界面，选择新提供方，填入新凭证，点保存即可。

**Q: 数据会被上传吗？**
A: 否，所有数据仅存储在本地设备 SharedPreferences 中。
