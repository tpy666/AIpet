🚀 快速开始
═══════════════════════════════════════════════════════════════════════════════

  1. 同步 Gradle
     $ ./gradlew build

  2. 运行应用
     $ ./gradlew installDebug
     或在 Android Studio 中 Shift+F10

  3. 预期效果
     ✓ 启动页 2 秒自动跳转
     ✓ 首页 3 个导航按钮
     ✓ 创建宠物并保存
     ✓ 查看宠物列表
     ✓ 与宠物聊天交互

🎯 功能流程
═══════════════════════════════════════════════════════════════════════════════

  SplashActivity (2秒延迟)
        ↓
  MainActivity (首页菜单)
        ├→ CreatePetActivity (创建宠物)
        │   └→ 保存到 SharedPreferences
        │       └→ 返回首页
        │
        ├→ PetCardListActivity (角色列表)
        │   ├→ 加载宠物列表
        │   └→ 点击卡片 → 进入聊天
        │
        └→ ChatActivity (聊天页面)
            ├→ 加载宠物信息
            ├→ 显示欢迎语
            ├→ 用户输入消息
            ├→ 本地模拟 AI 回复
            └→ 显示消息气泡
