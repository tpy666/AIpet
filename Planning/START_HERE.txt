## ✋ 快速问题排查

**问题: "Connection refused"**
→ 解决: 启动本地服务器 (run_server.bat)

**问题: "无法连接到模拟器"**
→ 解决: 确认 URL 使用 10.0.2.2:8000（不是 127.0.0.1）

**问题: "发送消息未收到回复"**
→ 解决: 查看日志 `adb logcat | grep ApiClient`

**问题: "Python 模块缺失"**
→ 解决: 运行 `pip install -r requirements.txt`

## 🎓 技术细节

### 本地服务工作原理
```
模拟器应用
    ↓
HTTP POST 到 http://10.0.2.2:8000/api/chat
    ↓
本地 Python Flask 服务器
    ↓
生成宠物个性化回复
    ↓
返回 JSON 响应
    ↓
应用显示回复
```

### 支持的 API 端点
- `GET /`                    服务器信息
- `GET /health`              健康检查
- `POST /api/chat`           本地聊天接口
- `POST /chat/completions`   OpenAI 兼容接口
