📱 使用流程
═══════════════════════════════════════════════════════════════════════════════

  场景 1: OpenAI 配置（生产）
  ───────────────────────────
  ① 获取 API Key
     https://platform.openai.com/account/api-keys
  ② 配置到应用
     ApiClient.configureOpenAI("sk-...", "gpt-3.5-turbo")
  ③ 发送消息
     用户输入 → API 调用 → 返回 → 显示
  ④ 如果失败
     API 错误 → 自动降级 → 本地模拟 → 显示

  场景 2: 本地后端（开发）
  ───────────────────────────
  ① 启动本地服务 (Python Flask)
     python app.py  # 运行在 localhost:8080
  ② 配置应用
     ApiClient.configureLocalBackend("http://192.168.1.100:8080")
  ③ 发送消息
     用户输入 → 本地 API → 返回 → 显示
  ④ 如果网络错误
     连接失败 → 重试 3 次 → 都失败 → 降级本地模拟

  场景 3: 离线模式（自动）
  ───────────────────────────
  ① 应用启动
     检查 API 配置 → 无效 → 使用本地模拟
  ② 用户发送消息
     提示信息 → 配置 API Key 以启用真实 AI
  ③ 成功响应
     本地生成回复 → 显示给用户
