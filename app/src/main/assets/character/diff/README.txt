Character emotion diff assets

Folders
- default/
- hoodie/
- formal/

Independent naming rule
- <outfitId>/<emotionTag>/<outfitId>__exp_<emotionTag>__v001.png

Examples
- default/normal/default__exp_normal__v001.png
- default/happy/default__exp_happy__v001.png
- hoodie/angry/hoodie__exp_angry__v001.png
- formal/sad/formal__exp_sad__v001.png

Supported emotionTag values in current app
- normal
- happy
- sad
- angry
- shy
- surprised
- excited

Place each file into the folder that matches outfitId and emotionTag.
